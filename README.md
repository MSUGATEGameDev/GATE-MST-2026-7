# Robot Warfare
**By:** Maddox Ross

### Instructor Thoughts
This game has a good atmosphere. It was fun to wander around the forest and fight enemies, and fun to think outside the box, and it was cool that you were able to import a new model for the player.

### Tweaks we made post-class
We fixed the part where the objectives didn’t reset at the start menu.

### Moving Forward
Working on getting the ninja model animated would be a good use of time, though it’s going to be a little complicated. You will have to find or make animations because they don’t appear to be pre-packaged with the ninja model. You’ll want to compare the existing player model’s animation names and make sure they match up with the names in the new animator.
Also, there are some places where your character can fall off the edge. You will probably want to make sure to have a kill player action triggered by an area trigger under the map  so they don’t just fall forever.
